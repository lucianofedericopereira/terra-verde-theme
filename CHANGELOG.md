# Changelog

All notable changes to Terra Verde will be documented here.

## [1.0.0] - 2026-02-18

### Added
- Initial release
- Full UI theme coverage: editor, sidebar, activity bar, status bar, tabs, panels, terminal, notifications, settings
- Syntax highlighting for JavaScript, TypeScript, HTML, CSS, Vue, PHP, Blade, JSON, YAML, TOML, Markdown and more
- Semantic token colors for all major token types
- Terminal ANSI palette matched to theme colors
- Git decoration colors
- Debug, test runner, and diff editor styling
- Bracket pair colorization
